# Business Analysis & Feature Validation

Perform comprehensive business analysis for: **$ARGUMENTS**

## Analysis Requirements

The Business Expert should provide:

### 1. Requirements Validation
- Clear business objective
- User value proposition
- Target user segments
- Pain points addressed
- Success metrics defined

### 2. Market Analysis
- Competitive landscape review
- Similar features in market
- Market demand validation
- Differentiation opportunities
- Market sizing (TAM/SAM/SOM)

### 3. ROI Calculation
**Investment**:
- Development time and cost
- Infrastructure costs
- Marketing/launch costs
- Total investment

**Returns**:
- Revenue impact (projected)
- Cost savings
- User growth impact
- Total expected returns

**Metrics**:
- ROI percentage
- Payback period
- Net present value (12 months)

### 4. User Story Creation
- User personas
- User stories with acceptance criteria
- User journey mapping
- Edge cases and error scenarios

### 5. Success Metrics & KPIs
**Product Metrics**:
- Engagement metrics (DAU, session duration)
- Retention metrics (Day 1, 7, 30)
- Adoption rate

**Business Metrics**:
- Revenue impact (MRR/ARR)
- User acquisition
- Conversion rates
- Customer satisfaction (NPS, CSAT)

### 6. Go-to-Market Strategy
- Target audience
- Positioning and messaging
- Launch plan (beta → public → growth)
- Marketing channels
- Success criteria

### 7. Risk Assessment
- Market risks
- Technical risks
- Resource risks
- Competitive risks
- Mitigation strategies

## Deliverables

- Business requirements document
- User stories with acceptance criteria
- ROI analysis report
- Competitive analysis
- Go-to-market plan
- Success metrics dashboard

Invoke the **business-expert** subagent to perform this analysis.
