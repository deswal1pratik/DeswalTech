## üí° How to Use

1. **Natural Language**: Just ask what you want to do
   - "Add a shadcn button component"
   - "Deploy this to Vercel"
   - "Create a Linear issue for this bug"

2. **MCPs Work Together**: Use multiple MCPs in one task
   - "Scrape this site (Supadata) and save to Supabase database"
   - "Create GitHub issue (GitHub) and add to Linear sprint (Linear)"

3. **Check Status**: Ask "What MCPs are available?" to see active ones
